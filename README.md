## Features

- Encrypts traffic via the Tor network for anonymity.
- Lightweight and easy to use.


## Requirements
- Tor installed and running
## Installation

Install TorLink

```bash
 git clone https://github.com/z0rok1t/TorLink.git

 cd TorLink

 make
  
 torlink <command> <URL>
```
